# Portify | Senior Developer Portfolio

A production-ready, high-converting React portfolio website designed for frontend and full-stack engineers. Built with a focus on technical depth, premium aesthetics, and responsive performance.

## üöÄ Teck Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: [Vercel](https://vercel.com/)

## ‚ú® Key Features

- **Premium Design**: Dark/Light/System theme support with smooth transitions.
- **Glassmorphism**: Modern UI cards with backdrop blur and subtle border effects.
- **Case Studies**: Detailed project sections focusing on Problem/Solution/Architecture.
- **Dynamic Animations**: Scroll-reveal effects and interactive hover states.
- **SEO Optimized**: Pre-configured metadata and Open Graph tags.
- **Responsive**: Mobile-first design that looks stunning on every screen size.

## üì¶ Deployment (Vercel)

The easiest way to deploy this portfolio is using [Vercel](https://vercel.com/):

1. Push your code to a GitHub/GitLab/Bitbucket repository.
2. Go to the [Vercel Dashboard](https://vercel.com/new).
3. Import your project.
4. Vercel will automatically detect Next.js and apply the correct build settings.
5. Click **Deploy**.
